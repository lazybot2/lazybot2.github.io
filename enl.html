触动版本校验出错

脚本供支持懒人源触动和官方触动，请到Cydia>>已安装列表>>找到触动，选择并卸载当前触动。
并到cydia添加https://apt.lazybot.vip懒人源。重新安装触动。
如果再次弹出此窗口,请到Cydia 已装列表中找到Substitute（Cydia substrate）后重启手机。
开机越狱后安装触动。
若还无法解决请解除越狱后再次越狱（ID机小心此操作）。

Bot supports LazyBot TS and official TS. Please go to Cydia>>Installed List>>to find the touch, select and uninstall the current TS.
And add to cydia https://apt.lazybot.vip Software source. Reinstall TS.
If this window pops up again, please find Substitute (Cydia substrate) in the Cydia installed list and restart the phone.
Install the touch after JB.
If it still cannot be solved, please rootfs and JB again (ID machine should be careful).
